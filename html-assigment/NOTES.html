The tag i choose are

Divison tag
The Divison tag defines a division or a section in an HTML document.
List tag
The List tag defines a list item
Navigations tab
The Navigation tag defines a set of navigation links.
Subscript tag
The Subscript tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like c2o.
Blockquote
The Blockquote tag specifies a section that is quoted from another source.
